# Data Cleaning and Preparation using Python

## Project Overview
This project focuses on cleaning and preparing a raw dataset for analysis. The dataset was inspected for missing values, duplicate records, invalid date formats, data type inconsistencies, and calculation errors.

## Objective
- Identify and handle missing values
- Detect and remove duplicate records
- Validate data formats and data types
- Verify data consistency
- Export a clean dataset for further analysis

## Tools and Libraries
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Tasks Performed

### 1. Dataset Inspection
- Checked dataset structure
- Reviewed columns and data types

### 2. Missing Value Analysis
- Identified missing values
- Calculated missing value percentages

### 3. Missing Value Treatment
- Replaced missing CouponCode values with "No Coupon"

### 4. Duplicate Record Detection
- Checked complete duplicate rows
- Checked duplicate Order IDs

### 5. Date Format Validation
- Converted Date column to datetime format
- Checked invalid dates

### 6. Data Type Validation
- Verified appropriate data types

### 7. Data Consistency Validation
- Verified TotalPrice = Quantity × UnitPrice

### 8. Final Data Quality Verification
- Confirmed no missing values
- Confirmed no duplicate records
- Confirmed no invalid dates
- Confirmed calculation consistency

### 9. Export Cleaned Dataset
- Exported cleaned dataset to Excel format

## Results
- Missing Values Handled Successfully
- Duplicate Rows Found: 0
- Duplicate Order IDs Found: 0
- Invalid Dates Found: 0
- Total Price Mismatches Found: 0
- Dataset Ready For Analysis
